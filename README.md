# TrackHire

A Java-based application that helps users manage and track job applications throughout the hiring process.

This project was built to practice real-world software development concepts such as Object-Oriented Programming (OOP), CRUD operations, data management, clean architecture, and backend development fundamentals.

---

## 📌 Features

- Add new job applications
- View all saved applications
- Update application status
- Delete applications
- Store important notes about applications
- Track application dates

---

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList Collections
- Java Scanner (User Input)
- JDBC & PostgreSQL/MySQL *(future upgrade)*
- Spring Boot REST API *(future upgrade)*

---

## 🧠 Concepts Practiced

This project demonstrates understanding of:

- Classes and Objects
- Encapsulation
- Separation of Concerns
- CRUD Operations
- Lists and Collections
- User Input Validation
- Method Design
- Basic Application Architecture

---

## 🏗️ Application Flow

User → Menu → Service Layer → Data Storage

1. User selects an option from the menu
2. The service layer processes the request
3. Data is stored and managed using collections
4. Updated results are displayed to the user

## ▶️ How to Run

### 1. Clone the Repository

```bash
git clone <repository-url>
```

### 2. Navigate into the Project

```bash
cd trackhire
```

### 3. Compile the Project

```bash
mvn compile
```

### 4. Run the Application

```bash
mvn exec:java
```
